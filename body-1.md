	XTAL1 和 XTAL2 分别是反相放大器的输入和输出。该反向放大器可以配置为片内振荡器。石晶振荡和陶瓷振荡均可采用。如果使用外部时钟源驱动设备，则不应连接 XTAL2。对内部时钟信号的多余输入需要通过两频分频触发，因此对外部时钟信号的脉冲宽度没有要求,但是必须保证脉冲的高低电平的所需宽度。

	8279采用单5V电源供电，40脚封装。

	VSS:地线输入线。
	### 3.3.2 8279 的编程方法
	8279按其功能可分为：键盘功能块； 显示功能块； 控制功能块； 接口和中央处理器功能块控制功能块包含控制和时间寄存器，定时和控制，扫描计数器三部分，它主要用于控制键盘的工作和显示功能块。
	控制和时间寄存器：用于存储中央单元的编程命令。 当8279中央处理器程序确定键盘和显示器的工作模式以及其他工作条件时，首先将命令控制数据放置在数据总线上，然后使A0 = 1，WR = 0CS = 0并将命令键保存在WR 沿边缘增加控制和时间寄存器，并进行解码以创建适当的功能。
	定时和控制：它包含一个基本的定时计数器。 第一个计数器是分频系数为2-31的预定时器。 可以通过程序预设分频系数，以使内部频率为100 kHz，这可能导致5.1 ms。 键盘扫描时间和10.3毫秒反跳时间。 其他计数器对该基本频率进行划分，并提供适当的键，行，键盘阵列扫描和显示扫描时间。
	扫描计数器：扫描计数器有两种工作模式。 在编码工作模式下，计数器提供一个二进制计数器。 在输出引脚SL0-SL3之后，可以为键盘提供扫描功能，并在外部解码后显示。 在此模式下，扫描计数器对最低有效两位进行解码，并且SL0-SL3输出1分之4的解码信号，以对显示器和键盘的扫描进行解码。
	键盘功能块包括: 返回缓冲区、键盘弹跳和控制、 8x8 FIFO 传感器随机存取存储器、 FIFO/传感器随机存取存储器状态。
	#### 返回缓冲器与键盘反跳及控制
	八个返回行由返回缓冲区缓冲。在键盘工作模式下，逐个检测这些行，找出行键中已关闭的键。如果回弹电路检测到钥匙关闭，则等待 10.3ms，然后记录钥匙是否仍然关闭。如果仍然关闭，则矩阵中 key 的行地址和列地址以及 SHIFT 和 CNTL 的状态一起发送到 FIFO 随机存取存储器,并且 FIFO 随机存取存储器中的数据格式如下:
	D7 D6 D5 D4 D3 D2 D1 D0
	CNTL SHIFT SCAN RET URN
	数据进行格式中,最高位CNTL,次高位为SHIFT状态,D5-D3来自不同扫描电子计数器,D2-D0来自中国返回一个计数器，扫描线计数器和回扫线计数器的值分别可以反映出被按下键的行，列的值，如果在传感器阵列中，返回行上的数据直接进入与正在扫描的阵列相对应的传感器随机存取存储器中的行，这样,每个开关的位置直接反映为一个传感器随机存取存储器的位置。
	#### FIFO/传感器随机存取存储器
	8x8 随机存取存储器是键盘模式和选通模式下的先进先出 (FIFO) 存储器。每条新的信息都是按顺序写的，然后按照写的顺序读出来。当传感器阵列扫描时，存储器被用作传感器随机存取存储器。此时，随机存取存储器中的每一行存储传感器阵列中相应行的状态。
	#### FIF0/传感器随机存取存储器状态
	在键盘或门控模式下，FIFO 状态跟踪 FIFO 中的字符数注意它是 “满” 还是 “空”，过多的写入或读取被认为是错误,当 FIFO 不为空时，状态逻辑提供一个中断应用程序 IRQ 信号。在传感器阵列扫描模式下，如果一个传感器改变，IRQ 是一个有效的高电平。FIFO 状态字的低 3 位表示 FIFO 中的字符数，F 表示 FIFO 随机存取存储器已满;O (超过 mn) 表示超限错误，即试图将另一个字符发送到完整的 FIFO;U (done 下) 表示 null 错误，即试图读取空 FIFO。S/F 有两种含义: 在传感器扫描模式下，S/F 是指传感器随机存取存储器中至少包含一个传感器闭合指示，在特殊错误模式下,S/F 位是一个错误标志，指示是否发生了多通道同时关闭错误,Du 位表示显示随机存取存储器尚未可用，因为 “清除显示” 或 “清除全部” 命令已完成其消除操作。
	显示功能块包括:显示寄存器,16X8显示随机存取存储器,显示地址寄存器)
	#### 显示随机存取存储器和显示寄存器

	### 3.4.1 温度数据采集电路
	DS18B20是Dallas Semiconductor Corporation的网络数字温度传感器。与其他的温度传感器相比，DS18B20具有以下功能：当连接到所述微处理器的唯一单线接口模式中，只有一个端口可用于实现微处理器和DS18B20之间的双向通信线路。
	DS18B20支持多点网络。几个DS18B20可以仅在三条线上并联连接，以实现联网的多点温度测量。 DS18B20不需要外围组件。所有传感器元件和转换电路都像晶体管一样集成在单个集成电路中。温度范围为55〜125°C，温度测量的固有分辨率为0.5°C。测量结果直接输出数字温度信号，并通过“初始化总线”串行发送到中央单元。同时，可以发送CRC校验码，具有较强的干扰和纠错功能。测量结果以9位数字模式串行传输。
	尽管DS18B20具有简单的温度测量系统，较高的温度测量精度，方便的连接以及较少的线路占用等优点，但在实际应用中还应考虑以下方面：
	1.尽管系统硬件很简单，但需要相对复杂的软件来进行补偿。由于DS18B20与微处理器之间的串行数据传输，因此必须保证在读写编程期间的读写时间，否则将无法读取温度测量结果。
	2. DS18B20的相关数据未提及连接到单个总线的DS18B20的数量。这很容易导致一个假设，即可以连接任意数量的DS18B20，而在实际应用中并非如此。如果一条总线上连接了8个以上的DS18B20，则有必要解决微处理器的总线驱动问题以及设计多点温度测量系统时要考虑的问题。
	3.长度限制是连接DS18B20的总线电缆。由于信号电缆的电子电缆的阻挡特性，如果距离太大，信号幅度会降低。如果在测试中普通信号电缆的传输长度超过50 m，则温度读取数据不正确。如果将总线电缆改为带屏蔽电缆的双绞线，则正常的通信距离可以达到150 m。

	### 3.4.3 发光二极管数码管
	发光二极管显示器即为发光二极管显示器，具有数据显示醒目、成本低、配置灵活、接口可以方便等特点，单片机控制应用管理系统设计中常用它来显示操作系统的工作生活状态和采集的信息网络输入数值等。
	根据发光二极管的排列结构不同，可将发光二极管分为 发光二极管数码管和 发光二极管点阵显示器。发光二极管数码管主要用于显示数字和一些字母和符号，发光二极管点阵显示器可以显示数字、字母、男人和图形等。发光二极管点阵显示虽然显示灵活，但它占用的单片机系统的软硬件资源比 发光二极管数码管要多得多。除了指定程序，大屏幕点阵显示器或LED显示器请求特殊的场合，几乎所有的微型计算机应用系统使用发光二极管数字显示。
	数字管显示有两种模式，包括静态显示模式和动态显示模式。静态显示程序非常不复杂，静态数据显示系统程序设计非常简单，占用CPU的时间很少。仅当显示字符更改时才调用显示程序。但是，硬件电路很多。每个数字管都需要一个8位I / O端口，一个8位驱动器和8个限流电阻。通常偶尔使用少量的数码管。发光二极管的静态显示器使用较少的组件，并且使用更多的数字电子管显示器进行切换很麻烦。为了简化电路并降低成本，系统使用了动态扫描显示方法。
	动态扫描显示模式的工作原理是分别打开每个显示器，即一次仅显示一位。为了使用户看到所有的显示器都已显示，必须加快每个显示器的照明周期（增加扫描频率）。通过使用人眼的视觉残留效果，可以实现与连续显示所有显示器相同的效果。动态扫描显示电路如图3.2.2所示：

	## 3.6 脉宽调制
	### 3.6.1 脉宽调制的介绍
	PWM 是脉冲宽度调制的英文缩写。高比的方波的时间段被称为占空比，例如，一秒钟高电平和的50％1第二占空比的PWM波的脉冲宽度调制（PWM）的低电平在开关电源中使用的术语。。这是根据稳压的控制方式进行分类的。除 PWM 型外，还有 PFM 型和 PWM 和 PFM 混合型。脉宽调制 (PWM) 开关稳压电路是在控制系统电路设计输出频率可以保持一个不变的情况下，通过电势差反馈来调节其占空比,从而能够达到社会稳定发展输出电势差的目的。
	脉冲宽度调制 (PWM) 是一种非常有效的利用微处理器的数字输出来控制模拟电路的技术，它被广泛用于从测量，控制和通信到电力转换许多领域。
	### 3.6.2 基本原理
	随着我国电子信息技术的发展，出现了多种 PWM 技术，包括: 相电势差控制 PWM 、脉冲信号宽度 PWM 方法、随机 PWM 、 SPWM 方法、线电势差控制 PWM 等,然而,镍氢动力电池使用智能手机充电器设计采用的脉冲时间宽度 PWM 方法，以等脉宽的脉冲串作为 PWM 波形，通过不断改变一个脉冲串的周期可进行分析调制,可以同时通过这些改变传统脉冲的宽度或占空比来调节电势差,并且我们可以选择通过研究采用学生适当的控制管理方法来实现协调地改变电势差和频率。
	模拟信号的值可以通过不断发展变化，对其时间和幅度的分辨率没有进行限制。9v 电池是一种模拟设备，因为它的输出电势差不完全等于 9v，而是随着时间而变化，并且可以取任何真实值。类似地，从电池吸收的电子流的强度不限于一组可能的值范围。模拟信号和数字信号的区别在于后者的值只能在预定的一组可能值内，例如，在 {0V 的集合中,5V}。
	模拟电势差和电子流的强度可直接用于控制，例如控制汽车收音机的音量。在简单的模拟收音机中，音量旋钮连接到可变电阻。拧动旋钮时，电阻值变大或变小；流过这个电阻的电子流的强度也增大或减小，从而改变驱动扬声器的电子流的强度值，使音量相应地变大或变小。与无线电一样，模拟电路的输出与输入成线性比例。
	在简单的模拟一个收音机中，音量控制旋钮连接到一些可变电阻。其中之一是模拟电路倾向于随时间漂移，因此难以调整。可以解决这个问题的精密模拟电路可以非常大，体积庞大 (如老式的家用立体声设备) 和昂贵。模拟电路还可能产生严重的热量，其功耗与工作元件两端的电势差与电子流的强度的乘积成正比。模拟电路也可能对噪声敏感，任何扰动或噪声都肯定会改变电子流的强度值。
	通过对模拟电路进行数字控制，可以大量系统的成本和功耗降低。此外，许多微控制器和 dsp 已经在中国芯片上包含了 PWM 控制器，这使得数字进行控制的实现自己变得可以更加具有容易。
	脉冲宽度调制信号的设计思想
	该对象的脉宽调制信号为方波，设定周期为1 s。 它的产生将时间计数器设置为10 ms时序，然后通过寄存器R3控制脉冲宽度调制信号的周期。 此主题仅具有模拟效果，并且对准确性没有过多要求，因此将1s周期划分为100。在相等的部分中，计时器设置为10 ms，计时器在R3中启动的频率为100。
	寄存器 R2 中存储的数据是根据检测电路和控制电路转换的数字。R2 中存储的值用于控制脉冲信号和 1s 内高电平的时间长度。这样我们可以从P2.6口检测到定周期进行脉冲可调的控制系统信号。
	### 3.6.4 脉宽调制信号的作用
	脉冲宽度调制信号从P3.0端口输出。 从P3.0端口输出的方波信号连接到三端双向可控硅开关的控制连接，以控制TRIAC的接通和关断。 如果方波在一段时间内以高电平出现，则双向可控硅的占空比变长，即加热元件散发的热量更多。 简而言之，加热元件释放的能级由方波在一个周期内变高的时间确定。
	### 3.6.5 脉冲宽度调制优点
	PWM 的好处的其中之一是从处理器到受控系统的信号是数字形式的，没有数模进行变换。保持数字形式的信号最大限度地减少噪声的影响。噪声的产生是因为只有在其强度足以阐述上升沿或下降沿跳变的时候，数字信号的稳定性才会被影响。
	除了指定程序，大屏幕点阵显示器或LED显示器请求特殊的场合，几乎所有的微型计算机应用系统使用发光二极管数字显示。模拟信号转换到 PWM信号 可是可以增加传输距离的长度。在接收端，可以滤除调制的高频方波，并通过适当的 RC 或 LC 网络将信号恢复为模拟形式。
	总之，PWM 不仅省钱、占地面积被节约、抗噪声性能强，而且是一种值得工程师在许多设计应用中使用的有效技术。
